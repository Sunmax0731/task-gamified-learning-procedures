# SKILL

## Start Order

1. Read README.md.
2. Read docs/source-idea-pack.json.
3. Review requirements, specification, design, implementation-plan, and test-plan.
4. Run `npm test`.
5. For visual checks, run `npm run serve -- --port 4173` and open `http://localhost:4173/app/`.

## Validation Contract

- Representative suite keeps happy-path, missing-required, warning, mixed-batch.
- QCDS means Quality, Cost, Delivery, Satisfaction; every grade must stay A- or above.
- `dist/task-gamified-learning-procedures-docs.zip` is generated by `npm test` and used as a release asset.
